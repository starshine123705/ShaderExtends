This is a Mod that provides Shader Model 5.0 Support for the game.
It currently support D3D11 and OpenGL.
And Only Vertex Shader/Pixel Shader and Compute Shader can be used.
It relies on my special shader compiler to compile the shader code, so it can compile once and used every where.
The Compiler only supports Windows now

The Mod is still in early development, so there are many features that are not implemented yet.
If any bugs tell me in Steam comments plz.

0.1.2.2 Update:
Fix the bug that destination set in Begin() isnt work in D3D11 Backend,and the bug that OpenGL backend cant work in Vertexless Shader. 
Change the Draw() method to support byte[] vertex data,allow you to customize the vertex buffer.

0.1.2 Update:
Fix the bug that cause the screen black when you leave the world, but caused UI cant be shadered.
Make the shader works like spritebatch,so it has a better user experience.

这个模组为游戏提供了Shader Model 5.0的支持。
目前支持D3D11和OpenGL。
并且只能使用顶点着色器/像素着色器和计算着色器。
它依赖于我编写的特殊着色器编译器来编译着色器代码，这使得它它可以编译一次并在任何地方使用。
编译器目前仅支持Windows。

该模组仍处于早期开发阶段，因此有许多功能尚未实现。
如果有任何错误，请在Steam评论中告诉我。

添加了配置以预览着色器效果

0.1.2.2 更新：
修复了在D3D11后端中Begin()中设置的目标不起作用的错误，以及OpenGL后端在无顶点着色器时无法工作的错误。
更改了Draw()方法以支持byte[]顶点数据，允许您自定义顶点缓冲区。

0.1.2更新：
修复了当你离开世界时导致屏幕变黑的错误，但导致UI无法被着色。
使着色器像spritebatch一样工作，因此具有更好的用户体验。

0.1.1更新：
添加了配置以预览着色器效果
